## 工具

### [python]发邮件工具(checkError.py)
* 利用smtp发送邮件
* 要打开邮箱的smtp服务（qq邮箱需要打开 163默认打开 eyou不支持smtp服务）
* 邮箱ssl认证（163.com qq.com）
* 中文乱码问题的处理

### [lua]解析xml文件(tmx_parser.lua)
* 文件类型：tmx文件, plist文件
* 依赖库  ：LuaXml, lfs 
* 版本	  ：lua 5.1 

### [lua]解析csv文件(csv_parser.lua)
* 文件类型：解析csv文件
* 依赖库  ：lfs 
* 版本	  ：lua 5.1
